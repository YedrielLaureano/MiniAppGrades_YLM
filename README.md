# MiniAppGrades_YLM

Sistema de notas, usando C++

## Description

Sistema de notas desarrollado en C++ que permite a los maestros gestionar la información de sus estudiantes. El sistema ofrece un menú interactivo con opciones para crear, editar, eliminar y ver notas (CRUD), además de registrar todas las acciones en un archivo de log para mantener autidoria.

## How to use

Clona el repositorio, compila el código y ejecuta el programa:

```bash
git clone https://github.com/YedrielLaureano/MiniAppGrades_YLM.git
cd MiniAppGrades_YLM
g++ src/main.cpp src/utils.cpp src/notas.cpp -o MiniAppGrades
./MiniAppGrades
